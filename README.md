# icon-label-grid

A polymer web component that shows a grid of icon-label components.

Example:
```html
        <icon-label-grid
          data="[[array]]">
        </icon-label-grid>
```

### Custom mixins:

```
        --icon-label-grid-cell-width (default 33%)
        --primary-text-color
        --secondary-text-color
```

### Dependencies

Dependencies are installed using [Bower](http://bower.io/):

    npm install -g bower
    bower install
